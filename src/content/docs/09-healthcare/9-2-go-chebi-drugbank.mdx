SNOMED CT와 ICD가 임상 의료 영역을 다룬다면, Gene Ontology, ChEBI, DrugBank는 기초과학과 제약 연구의 표준 지식 자원이다. 이 세 온톨로지·데이터베이스는 신약 발굴, 유전체 연구, 시스템 생물학에서 핵심 참조 자원으로 활용된다.

## Gene Ontology (GO)

Gene Ontology는 1998년 시작된 생물정보학 커뮤니티 프로젝트로, 유전자와 유전자 산물(단백질, RNA)의 기능을 표준화된 어휘로 기술한다. OBO(Open Biomedical Ontologies) 형식과 OWL 형식으로 배포된다.

### 세 서브온톨로지

```text
Gene Ontology 3개 측면 (서브온톨로지)
──────────────────────────────────────────────────────────
 ┌─────────────────────────────────────────────────────┐
 │              GO:0003674 molecular_function           │
 │  ┌─────────────────────────────────────────────┐   │
 │  │ GO:0005488 binding                           │   │
 │  │   └─ GO:0005515 protein binding             │   │
 │  │ GO:0003824 catalytic activity               │   │
 │  │   └─ GO:0004672 protein kinase activity     │   │
 │  └─────────────────────────────────────────────┘   │
 └─────────────────────────────────────────────────────┘

 ┌─────────────────────────────────────────────────────┐
 │              GO:0008150 biological_process           │
 │  ┌─────────────────────────────────────────────┐   │
 │  │ GO:0009987 cellular process                  │   │
 │  │   └─ GO:0007049 cell cycle                  │   │
 │  │ GO:0065007 biological regulation            │   │
 │  │   └─ GO:0006915 apoptotic process           │   │
 │  └─────────────────────────────────────────────┘   │
 └─────────────────────────────────────────────────────┘

 ┌─────────────────────────────────────────────────────┐
 │              GO:0005575 cellular_component           │
 │  ┌─────────────────────────────────────────────┐   │
 │  │ GO:0005622 intracellular                     │   │
 │  │   ├─ GO:0005634 nucleus                     │   │
 │  │   └─ GO:0005737 cytoplasm                   │   │
 │  └─────────────────────────────────────────────┘   │
 └─────────────────────────────────────────────────────┘

  유전자 산물(단백질)은 세 측면 모두에서 어노테이션 가능
  예: TP53 → MF: DNA binding / BP: apoptosis / CC: nucleus
```

GO는 세 개의 독립적인 DAG(Directed Acyclic Graph) 온톨로지로 구성된다.

**Molecular Function (MF):** 유전자 산물이 수행하는 분자 수준의 활동.
예: `GO:0004672` (protein kinase activity), `GO:0005488` (binding)

**Biological Process (BP):** 유전자 산물이 참여하는 더 큰 생물학적 과정.
예: `GO:0006915` (apoptotic process), `GO:0007049` (cell cycle)

**Cellular Component (CC):** 유전자 산물이 활성화되는 세포 내 위치.
예: `GO:0005737` (cytoplasm), `GO:0005634` (nucleus)

세 온톨로지는 서로 다른 측면을 기술하므로, 하나의 유전자 산물이 MF, BP, CC 각각에서 여러 GO 용어와 연결될 수 있다.

### GO 관계 유형

GO 온톨로지에서 사용되는 주요 관계:
- `is_a`: 클래스 계층 관계. 특수 → 일반
- `part_of`: 구성 요소 관계. 핵소체 part_of 핵
- `regulates` / `positively_regulates` / `negatively_regulates`: 조절 관계
- `has_part`: 역 part_of

### GO 어노테이션

GO 용어 자체보다 더 중요한 것이 GO 어노테이션이다. 특정 유전자(예: UniProt P04637, TP53)와 GO 용어를 연결하는 큐레이션 데이터다. 각 어노테이션에는 증거 코드(Evidence Code)가 포함된다.

- `EXP`: 실험적 증거
- `IDA`: 직접 분석(Direct Assay)에 의한 증거
- `IEA`: 전자적 추론(Inferred from Electronic Annotation) — 자동화, 신뢰도 낮음
- `ISS`: 서열 유사성에 의한 추론

GO는 SPARQL 엔드포인트를 제공하며, UniProt, Ensembl, NCBI Gene 데이터베이스와 연계된다.

### 활용 사례

**유전자 세트 풍부화 분석(GSEA):** RNA-seq 실험에서 차등 발현 유전자 목록을 GO 용어와 대조해 어떤 생물학적 과정이 통계적으로 풍부하게 나타나는지 분석한다.

**기능 예측:** 기능이 알려지지 않은 단백질의 GO 용어를 서열 유사성이나 머신러닝으로 예측한다.

## ChEBI (Chemical Entities of Biological Interest)

ChEBI는 EBI(European Bioinformatics Institute)가 관리하는 화학 물질 온톨로지다. 생물학적으로 관련 있는 소분자, 이온, 작용기를 계층적으로 분류한다.

### 구조: 구조 vs 역할

ChEBI의 특징적인 설계는 화학 물질의 **구조(structure)**와 **역할(role)**을 분리하는 것이다.

**구조 계층:** 화학적 구조에 따른 분류.
예: `아스피린` is_a `아세틸살리실산류` is_a `살리실산류` is_a `벤조산류`

**역할 계층:** 생물학적·약학적 맥락에서의 역할.
예: `아스피린` has_role `진통제`, `항혈소판제`, `COX 억제제`

동일한 화학 물질이 구조 계층에서는 하나의 위치를 차지하지만, 역할 계층에서는 여러 역할을 가질 수 있다.

### ChEBI 식별자

모든 ChEBI 개념은 `CHEBI:` 접두어로 시작하는 숫자 ID를 가진다. 예: `CHEBI:15422` (ATP), `CHEBI:15854` (아스피린).

### 연계 데이터베이스

ChEBI는 PubChem, ChemSpider, KEGG, Reactome, UniProt과 크로스-레퍼런스를 제공한다. SPARQL 엔드포인트를 통해 접근할 수 있다.

## DrugBank

DrugBank는 캐나다 앨버타 대학에서 개발·유지하는 약물 정보 데이터베이스다. 약물의 화학적 정보, 약리학, 약물-표적 상호작용, 약물-약물 상호작용을 통합한 지식그래프다.

### 데이터 구조

DrugBank의 핵심 엔티티:

**Drug:** 승인 약물, 임상시험 약물, 뉴트리서포트 등 14,000개 이상의 약물 항목.
- 화학 구조 (SMILES, InChI)
- 약동학·약력학 정보
- 적응증, 금기, 부작용

**Target:** 약물이 작용하는 단백질 표적. UniProt ID로 연결.
- 작용 메커니즘 (억제, 활성화, 조절)
- 결합 부위 정보

**Pathway:** 약물이 관여하는 대사·신호 경로. KEGG, Reactome과 연계.

**Interaction:** 약물-약물 상호작용 데이터. 상호작용 유형(약동학적, 약력학적)과 임상적 유의성 수준.

### 지식그래프로의 활용

DrugBank XML 데이터를 RDF로 변환해 SPARQL 질의할 수 있다. Bio2RDF 프로젝트가 DrugBank 포함 여러 생명과학 DB를 RDF Linked Data로 변환해 제공했다.

```sparql
# 당뇨병 치료에 사용되는 약물과 그 표적 단백질 조회
SELECT ?drug ?drugName ?target ?targetName WHERE {
  ?drug a db:Drug ;
        db:name ?drugName ;
        db:indication ?indication ;
        db:target ?target .
  ?target db:name ?targetName .
  FILTER(CONTAINS(LCASE(?indication), "type 2 diabetes"))
}
```

## 세 온톨로지의 연계: 신약 발굴 파이프라인

세 자원은 개별적으로도 유용하지만 통합될 때 더 강력한 분석이 가능하다.

**약물 재목적화(Drug Repurposing) 시나리오:**

1. GO 어노테이션: 알츠하이머 관련 유전자 목록 식별 (APOE, APP, PSEN1)
2. DrugBank: 해당 유전자 산물을 표적으로 하는 기존 약물 조회
3. ChEBI: 조회된 약물의 화학 구조 유사성으로 새 후보 물질 탐색
4. SNOMED CT: 해당 약물의 현재 적응증과 금기 확인

이 파이프라인은 실험 없이 기존 지식만으로 새로운 약물 후보를 제안한다. KG 기반 약물 재목적화는 신약 발굴 비용과 시간을 줄이는 유망한 접근법으로 연구되고 있다.

| 온톨로지 | 관리 기관 | 주요 대상 | 형식 | SPARQL |
|---------|---------|---------|------|--------|
| Gene Ontology | GO Consortium | 유전자 기능 | OBO/OWL | 제공 |
| ChEBI | EBI | 화학 물질 | OBO/OWL | 제공 |
| DrugBank | 앨버타 대학 | 약물 정보 | XML/RDF | 제한적 |
